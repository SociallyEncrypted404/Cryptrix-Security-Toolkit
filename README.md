# Void://Cryptrix Security - Advanced Threat Scanner

A Windows batch-based security toolkit designed to automate defensive security checks and threat detection.

## Features

- Microsoft Defender malware scan integration
- Suspicious file detection
- Startup persistence analysis
- Running process inspection
- System logging similar to /var/log
- Modular design
- Lightweight and easy to run

## Modules

virus_scan.bat
Uses Microsoft Defender engine to scan for known threats.

malware_hunter.bat
Searches system for suspicious file types commonly used in malware.

startup_scan.bat
Checks common persistence locations used by malicious software.

process_scan.bat
Analyzes running processes for suspicious behavior.

## Log Output

Scan results are saved inside the logs folder for review.
